#  Hotel Booking Management System

A simple **Java CLI application** developed during my **Java Programming Internship at SyntecxHub**.  
The project manages hotel room bookings, cancellations, and room availability using Java OOP concepts.

---

##  Features
- View available and booked rooms
- Book rooms by ID or type
- Cancel bookings
- Data persistence using CSV file
- Clean OOP design and colorful CLI

---

##  Tech Stack
- Java 21
- Collections (ArrayList)
- File I/O (CSV)
- Exception Handling
